<h3>动态sql</h3>
<p>适用于模糊查询，动态添加查询条件，使用<code>&#x3C;where></code>标签，其中用<code>&#x3C;if></code>标签做判断，用法如下</p>
<p>UserDao.xml</p>
<pre><code class="language-xml">    &#x3C;select id="selectAll" resultMap="BaseResultMap">
        select * from t_user
        &#x3C;where>
            &#x3C;if test="userId != null and userId != ''">and user_id like '%${userId}%'&#x3C;/if>
            &#x3C;if test="userName != null">and user_name like '%${userName}%'&#x3C;/if>
        &#x3C;/where>
    &#x3C;/select>
</code></pre>

<h3>选择标签</h3>
<p><code>&#x3C;where>&#x3C;choose>&#x3C;when>&#x3C;otherwise></code>相当于 <code>if...else if... else...</code></p>
<pre><code class="language-xml">    &#x3C;select id="selectAllByOp" resultType="com.zr.vo.User">
        select * from t_user
        &#x3C;where>
            &#x3C;choose>
                &#x3C;when test="op=='All'">user_name = #{userName}&#x3C;/when>
                &#x3C;when test="op=='Left'">user_name like '${userName}%'&#x3C;/when>
                &#x3C;when test="op=='Right'">user_name like '%${userName}'&#x3C;/when>
                &#x3C;otherwise>user_name like '%${userName}%'&#x3C;/otherwise>
            &#x3C;/choose>
        &#x3C;/where>
    &#x3C;/select>
</code></pre>
<p><code>test='判断语句'</code> 作为判断条件，满足则加入标签内的语句</p>
<h3>trim标签</h3>
<p>mybatis的trim标签一般用于去除sql语句中多余的and关键字，逗号，或者给sql语句前拼接 “where“、“set“以及“values(“ 等前缀，或者添加“)“等后缀，可用于选择性插入、更新、删除或者条件查询等操作。</p>
<p>示例</p>
<pre><code class="language-xml">    &#x3C;insert id="insert">
&#x3C;!--        insert into t_user values (#{userId}, #{userName}, #{password}, #{sex}, #{tel}, #{status}, #{img}, #{deptId} #{createTime}, #{createUser}, #{modifyTime}, #{modifyUser})-->
        insert into t_user
        &#x3C;trim prefix="(" suffix=")" suffixOverrides=","> &#x3C;!--忽略后缀逗号 -->
            &#x3C;if test="userId != null">user_id,&#x3C;/if>
            &#x3C;if test="userName != null">user_name,&#x3C;/if>
            &#x3C;if test="password!= null">password,&#x3C;/if>
            &#x3C;if test="sex != null">sex,&#x3C;/if>
            &#x3C;if test="tel != null">tel,&#x3C;/if>
            &#x3C;if test="status!= null">status,&#x3C;/if>
            &#x3C;if test="img != null">img,&#x3C;/if>
            &#x3C;if test="deptId != null">dept_id,&#x3C;/if>
            &#x3C;if test="createTime!= null">create_time,&#x3C;/if>
            &#x3C;if test="createUser!= null">create_user,&#x3C;/if>
            &#x3C;if test="modifyTime!= null">modify_time,&#x3C;/if>
