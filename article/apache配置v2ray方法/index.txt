    <html><head></head><body><h2>配置 Apache</h2>
<p>我们假设，你已经有了一个可以正常运行的V2Ray环境和Apache2。</p>
<h3>1. 在服务器上开启以下 Apache 模组</h3>
<pre><code>sudo a2enmod ssl
sudo a2enmod proxy
sudo a2enmod proxy<span class="hljs-emphasis">_wstunnel
sudo a2enmod proxy_</span>http
sudo a2enmod rewrite
sudo a2enmod headers
</code></pre>
<h3>2. 修改Apache 配置文件</h3>
<p>我们找到配置文件，一般在 /etc/apache2 文件夹下。 一般，我们可以在该目录下找到sites-available（可用的配置文件） 和 sites-enabled（启用的配置文件） 我们进入 sites-enabled，找到 443（即 HTTPS 配置文件，例如：000-default-le-ssl.conf）。 把以下配置加到<code><span class="hljs-section">&lt;VirtualHost&gt;</span><span class="hljs-section">&lt;/VirtualHost&gt;</span></code>之间</p>
<pre><code><span class="hljs-section">&lt;LocationMatch <span class="hljs-string">"/{ws_path}}/"</span>&gt;</span>
<span class="hljs-attribute">ProxyPass</span> ws://<span class="hljs-number">127.0.0.1</span>:{port}/{ws_path}/ upgrade=WebSocket
<span class="hljs-attribute">ProxyAddHeaders</span> <span class="hljs-literal">Off</span>
<span class="hljs-attribute">ProxyPreserveHost</span> <span class="hljs-literal">On</span>
<span class="hljs-attribute">RequestHeader</span> set Host <span class="hljs-variable">%{HTTP_HOST}</span>s
<span class="hljs-attribute">RequestHeader</span> set X-Forwarded-For <span class="hljs-variable">%{REMOTE_ADDR}</span>s
<span class="hljs-section">&lt;/LocationMatch&gt;</span>
</code></pre>
