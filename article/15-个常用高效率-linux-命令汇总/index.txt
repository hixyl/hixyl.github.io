    <html><head></head><body><p>![15 个常用高效率 Linux 命令汇总]</p>
<p>本文提供了 Linux 中最常用的 15 个命令的常用用法。</p>
<p>这并不全面，但可以让你快速了解一些常见的 Linux 命令，从而一只脚跨入 Linux 的大门。</p>
<p>如果你觉得还有其他常用的 Linux 命令，可以在评论区补充。</p>
<h2>1 grep</h2>
<p><strong>在文件中搜索给定字符串</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>grep -i -n -w -C <span class="hljs-number">3</span> <span class="hljs-string">"the"</span> filename
</code></pre>
<p>-i：不区分大小写</p>
<p>-n：打印行号</p>
<p>-w：只输出完整单词（比如只考虑 the，不考虑 they）</p>
<p>-C：指定显示匹配项前后的行数</p>
<p><strong>根据特定模式匹配，在一系列文件中搜索</strong>：</p>
<pre><code class="language-text">$ grep <span class="hljs-string">"this"</span> demo_*
demo_file:this <span class="hljs-built_in">line</span> is <span class="hljs-keyword">the</span> <span class="hljs-number">1</span>st <span class="hljs-built_in">lower</span> <span class="hljs-keyword">case</span> <span class="hljs-built_in">line</span> <span class="hljs-keyword">in</span> this <span class="hljs-built_in">file</span>.
demo_file1:Two <span class="hljs-keyword">lines</span> above this <span class="hljs-built_in">line</span> is <span class="hljs-literal">empty</span>.
demo_file2:And this is <span class="hljs-keyword">the</span> <span class="hljs-keyword">last</span> <span class="hljs-built_in">line</span>.
</code></pre>
<p>使用 <code>demo_*</code> 匹配到 <code><span class="hljs-attribute">demo_file</span></code>, <code>demo_file1</code>, <code>demo_file2</code>三个文件。</p>
<p><strong>匹配文件中的正则表达式</strong>：</p>
<pre><code class="language-text">$ <span class="hljs-keyword">grep</span> <span class="hljs-string">"want.*go"</span> demo_file
They want <span class="hljs-keyword">to</span> <span class="hljs-keyword">go</span> shopping.
I want <span class="hljs-keyword">to</span> <span class="hljs-keyword">go</span> swimming.
</code></pre>
<p>grep 还支持使用正则表达式来搜索字符串，比如上面的语句就表示搜索以 <code><span class="hljs-attribute">want</span></code> 开头，以 <code><span class="hljs-attribute">go</span></code> 结尾的模式。</p>
<p>常用正则表达式可以查看这篇文章。</p>
<p>[程序员阿德：超简单正则表达式入门教程29 赞同 · 2 评论文章![]</p>
<p><strong>递归搜索所有文件</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>grep -r <span class="hljs-string">"the"</span> *
</code></pre>
<p><strong>反向匹配，即只显示没有匹配上的行</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>grep -v <span class="hljs-string">"the"</span> filename
</code></pre>
<h2>2 find</h2>
<p><strong>使用文件名查找文件</strong>：</p>
<pre><code class="language-text">$ <span class="hljs-built_in">find</span> -name test.cpp
</code></pre>
<p><strong>使用文件名查找文件</strong>（不区分大小写）：</p>
<pre><code class="language-text">$ <span class="hljs-built_in">find</span> -iname test.cpp
</code></pre>
<p><strong>指定查找路径</strong>：</p>
<pre><code class="language-text">$ <span class="hljs-built_in">find</span> dirname/ -iname test.cpp
</code></pre>
<p><strong>反向匹配，即查找不是指定名称的文件和目录</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> find <span class="hljs-operator">-not</span> <span class="hljs-literal">-iname</span> test.cpp
</code></pre>
<p><strong>根据文件类型查找文件</strong>：</p>
<p>需要使用 <code>-<span class="hljs-keyword">type</span></code> 选项，只查找普通文件(f)，只查找所有目录(d)。</p>
<pre><code class="language-text">$ find . -<span class="hljs-keyword">type</span> <span class="hljs-type">f
</span></code></pre>
<p><strong>查找最大的 5 个文件</strong>：</p>
<pre><code class="language-text"><span class="hljs-meta prompt_">$ </span><span class="language-bash">find . -<span class="hljs-built_in">type</span> f -<span class="hljs-built_in">exec</span> <span class="hljs-built_in">ls</span> -s {} \;  <span class="hljs-built_in">sort</span> -n -r  <span class="hljs-built_in">head</span> -5</span>
</code></pre>
<p>想查找最小的文件，将 <code><span class="hljs-deletion">-r</span></code> 去掉即可。</p>
<h2>3 ps</h2>
<p><strong>查看当前正在运行的进程信息</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">ps</span> <span class="hljs-literal">-ef</span>  more
</code></pre>
<p><strong>以树状结构查看当前正在运行的进程信息</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">ps</span> <span class="hljs-literal">-efH</span>  more
</code></pre>
<p><code><span class="hljs-deletion">-H</span></code>：代表进程层次结构</p>
<h2>4 free</h2>
<p>显示系统中空闲内存、已用内存、交换内存等内存信息。默认输出以 KB 为单位显示。</p>
<pre><code class="language-text">$ free
              total        used        free      shared  buff/cache   available
Mem:        <span class="hljs-number">8208948</span>     <span class="hljs-number">6610388</span>     <span class="hljs-number">1369208</span>       <span class="hljs-number">17720</span>      <span class="hljs-number">229352</span>     <span class="hljs-number">1464828</span>
Swap:      <span class="hljs-number">25165824</span>     <span class="hljs-number">1199792</span>    <span class="hljs-number">23966032</span>
</code></pre>
<p>如果认为字节显示不方便，可以通过指定参数选择其他单位：-g 表示GB，-k 表示KB，-m 表示MB。</p>
<p><strong>查看总内存（包括交换内存）</strong>：</p>
<pre><code class="language-text">$ free -t
              total        used        free      shared  buff/cache   available
Mem:        <span class="hljs-number">8208948</span>     <span class="hljs-number">6425468</span>     <span class="hljs-number">1554128</span>       <span class="hljs-number">17720</span>      <span class="hljs-number">229352</span>     <span class="hljs-number">1649748</span>
Swap:      <span class="hljs-number">25165824</span>     <span class="hljs-number">1202472</span>    <span class="hljs-number">23963352</span>
Total:     <span class="hljs-number">33374772</span>     <span class="hljs-number">7627940</span>    <span class="hljs-number">25517480</span>
</code></pre>
<h2>5 df</h2>
<p>显示文件系统磁盘空间使用情况，默认情况下 <code><span class="hljs-built_in">df</span> -k</code> 以字节为单位显示输出。</p>
<pre><code class="language-text">$ df -k
Filesystem           <span class="hljs-number">1</span>K-blocks      Used Available Use% Mounted on
/dev/sda1             <span class="hljs-number">29530400</span>   <span class="hljs-number">3233104</span>  <span class="hljs-number">24797232</span>  <span class="hljs-number">12</span>% /
/dev/sda2            <span class="hljs-number">120367992</span>  <span class="hljs-number">50171596</span>  <span class="hljs-number">64082060</span>  <span class="hljs-number">44</span>% /home
</code></pre>
<p><code><span class="hljs-built_in">df</span> -h</code> 以我们更方便阅读的形式显示输出，即大小将以 GB 显示。</p>
<pre><code class="language-text">$ df -h
Filesystem            <span class="hljs-keyword">Size</span>  Used Avail Use% Mounted on
<span class="hljs-regexp">/dev/</span>sda1              <span class="hljs-number">29</span>G  <span class="hljs-number">3.1</span>G   <span class="hljs-number">24</span>G  <span class="hljs-number">12</span>% /
<span class="hljs-regexp">/dev/</span>sda2             <span class="hljs-number">115</span>G   <span class="hljs-number">48</span>G   <span class="hljs-number">62</span>G  <span class="hljs-number">44</span>% /home
</code></pre>
<p>使用 -T 选项显示文件系统的类型。</p>
<pre><code class="language-text">$ df -T
Filesystem    Type   <span class="hljs-number">1</span>K-blocks      Used Available Use% Mounted on
/dev/sda1     ext4    <span class="hljs-number">29530400</span>   <span class="hljs-number">3233120</span>  <span class="hljs-number">24797216</span>  <span class="hljs-number">12</span>% /
/dev/sda2     ext4   <span class="hljs-number">120367992</span>  <span class="hljs-number">50171596</span>  <span class="hljs-number">64082060</span>  <span class="hljs-number">44</span>% /home
</code></pre>
<h2>6 tar</h2>
<p><strong>创建一个 tar 压缩包</strong>：</p>
<pre><code class="language-text"><span class="hljs-meta prompt_">$ </span><span class="language-bash">tar cvf archive_name.tar <span class="hljs-built_in">dirname</span>/</span>
</code></pre>
<p>-c：创建一个新的压缩包</p>
<p>-v：压缩时详细列出文件名（可选）</p>
<p>-f：指定压缩包名</p>
<p><strong>从现有的 tar 压缩包中提取</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>tar xvf archive_name.tar
</code></pre>
<p><strong>查看 tar 压缩包中的文件</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>tar tvf archive_name.tar
</code></pre>
<h2>7 kill</h2>
<p>使用 kill 命令终止进程。首先使用 ps -ef 命令获取进程 id，然后使用 kill -9 杀死正在运行的 Linux 进程。</p>
<pre><code class="language-text">$ <span class="hljs-keyword">ps</span> -ef  <span class="hljs-keyword">grep</span> <span class="hljs-keyword">vim</span>
ramesh    <span class="hljs-number">7243</span>  <span class="hljs-number">7222</span>  <span class="hljs-number">9</span> <span class="hljs-number">22</span>:<span class="hljs-number">43</span> <span class="hljs-keyword">pts</span>/<span class="hljs-number">2</span>    <span class="hljs-number">00</span>:<span class="hljs-number">00</span>:<span class="hljs-number">00</span> <span class="hljs-keyword">vim</span>

$ kill -<span class="hljs-number">9</span> <span class="hljs-number">7243</span>
</code></pre>
<h2>8 rm</h2>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">rm</span> <span class="hljs-literal">-i</span> filename.txt
</code></pre>
<p>-i：在删除文件之前获得确认（非常有用的一个参数，防止误删）</p>
<p>递归删除目录下的所有文件和目录：</p>
<pre><code class="language-text"><span class="hljs-meta prompt_">$ </span><span class="language-bash"><span class="hljs-built_in">rm</span> -rf <span class="hljs-built_in">dirname</span>/</span>
</code></pre>
<p>-r：递归删除</p>
<p>-f：强制删除</p>
<h2>9 cp</h2>
<p>将 file1 复制到 file2，并保留模式、所有权和时间戳。</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">cp</span> <span class="hljs-literal">-p</span> file1 file2
</code></pre>
<p>将 file1 复制到 file2，如果 file2 存在，则在覆盖前提示确认。</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">cp</span> <span class="hljs-literal">-i</span> file1 file2
</code></pre>
<h2>10 mv</h2>
<p>移动文件到新路径：</p>
<pre><code class="language-text"><span class="hljs-meta prompt_">$ </span><span class="language-bash"><span class="hljs-built_in">mv</span> -vi filename <span class="hljs-built_in">dirname</span>/</span>
</code></pre>
<p>将 file1 重命名为 file2，如果 file2 存在，则在覆盖前提示确认：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">mv</span> <span class="hljs-literal">-vi</span> file1 file2
</code></pre>
<p>-i：覆盖前提示确认</p>
<p>-f：不提示直接覆盖</p>
<p>-v：显示文件重命名过程</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">mv</span> <span class="hljs-literal">-v</span> <span class="hljs-number">1</span>.txt <span class="hljs-number">01</span>/
renamed <span class="hljs-string">'1.txt'</span> -&gt; <span class="hljs-string">'01/1.txt'</span>
</code></pre>
<h2>11 cat</h2>
<p>你可以在不打开文件的情况下，查看文件内容，甚至可以同时查看多个文件内容：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">cat</span> file1 file2
</code></pre>
<p><strong>查看文件内容时，显示行号</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> <span class="hljs-built_in">cat</span> <span class="hljs-literal">-n</span> filename
</code></pre>
<p><strong>查看文件前两行</strong>：</p>
<pre><code class="language-text"><span class="hljs-built_in">cat</span> -n filename  <span class="hljs-built_in">head</span> -n 2
</code></pre>
<p><strong>查看文件后两行</strong>：</p>
<pre><code class="language-text"><span class="hljs-built_in">cat</span> -n filename  <span class="hljs-built_in">tail</span> -n 2
</code></pre>
<p><strong>三种方法显示文件10行到50行内容</strong>：</p>
<pre><code class="language-text"><span class="hljs-built_in">cat</span> filename  <span class="hljs-built_in">head</span> -n 50  <span class="hljs-built_in">tail</span> -n +10
<span class="hljs-built_in">cat</span> filename  <span class="hljs-built_in">tail</span> -n +10  <span class="hljs-built_in">head</span> -n 41
sed -n <span class="hljs-string">'100,500p'</span> filename
</code></pre>
<h2>12 tail</h2>
<p>查看文件最后 n 行（默认 n 为 10）：</p>
<pre><code class="language-text">tail -<span class="hljs-built_in">n</span> <span class="hljs-built_in">N</span> filename
</code></pre>
<p>实时查看文件内容：</p>
<pre><code class="language-text"><span class="hljs-variable">$</span> tail <span class="hljs-operator">-f</span> filename
</code></pre>
<p>这对于查看不断增长的日志文件很有用。可以使用 CTRL-C 终止该命令。</p>
<h2>13 less</h2>
<p>less 在查看巨大的日志文件时非常高效，因为它不需要在打开时加载完整的文件。</p>
<pre><code class="language-text"><span class="hljs-keyword">less</span> filename
</code></pre>
<h2>14 ifconfig</h2>
<p>使用 ifconfig 命令查看或配置 Linux 系统上的网络接口。</p>
<p><strong>查看所有网络接口以及状态</strong>：</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>ifconfig -a
</code></pre>
<p><strong>启动或停止特定网络接口</strong>。</p>
<pre><code class="language-text"><span class="hljs-variable">$ </span>ifconfig eth0 up
<span class="hljs-variable">$ </span>ifconfig eth0 down
</code></pre>
<h2>15 shutdown</h2>
<p><strong>关闭系统并立即关闭电源</strong>：</p>
