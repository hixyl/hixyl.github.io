<h3>router 标签</h3>
<p>router中常用的标签为</p>
<ol>
<li>router-link：路由连接标签，最终被解析为a标签。</li>
<li>router-view：路由插槽，用于视图显示区域。</li>
</ol>
<p>router的跳转方式：
• push： 向history 栈添加一个新的记录，可以使用浏览器的back按键返回。<br>
• replace：直接替换当前浏览器路径，无法返回。</p>
<p>• $router : 是路由操作对象，提供了路由切换参数传递等功能。<br>
• $route : 路由信息对象，只读对象，可以读取当前组件路由信息。</p>
<p><img src="md04.PNG" alt="alt"></p>

<h2>导航守卫</h2>
<p>全局路由守卫：<code>beforeEach</code><br>
守卫的参数：每个守卫函数都具备如下三个参数，通过参数可以获取跳转前、跳转后的路由对象。<br>
• to：要进入的目标路由对象<br>
• from：正在离开的路由对象<br>
• next：函数，只有调用该函数，导航钩子才会被resolved</p>
<p>main.js</p>
<pre><code class="language-javascript">// The Vue build version to load with the `import` command
// (runtime-only or standalone) has been set in webpack.base.conf with an alias.
import Vue from 'vue'
import App from './App'
import router from './router'

Vue.config.productionTip = false

//全局路由守卫
router.beforeEach((to, from, next) => {
  console.info(to.name, from.name)
  next()//执行组件切换
  // to and from are both route objects. must call `next`.
})
/* eslint-disable no-new */
new Vue({
  el: '#app',
  router,
  components: { App },
  template: '&#x3C;App/>'
})
</code></pre>
