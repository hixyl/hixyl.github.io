<h4 id="题目描述"><a href="#题目描述">题目描述</a></h4>
<p>给你一棵二叉树的根节点 <code>root</code> ，返回该树的 <strong>直径</strong> 。</p>
<p>二叉树的 <strong>直径</strong> 是指树中任意两个节点之间最长路径的 <strong>长度</strong> 。这条路径可能经过也可能不经过根节点 <code>root</code> 。</p>
<p>两节点之间路径的 <strong>长度</strong> 由它们之间边数表示。</p>
<hr>
<h4 id="示例"><a href="#示例">示例</a></h4>
<p>示例 1：
<img src="https://assets.leetcode.com/uploads/2021/03/06/diamtree.jpg" alt=""></p>
<blockquote>
<p>输入： root = [1,2,3,4,5]</p>
<p>输出： 3</p>
<p>解释： 3 ，取路径 [4,2,1,3] 或 [5,2,1,3] 的长度。</p>
</blockquote>
<p><strong>示例 2：</strong></p>
<blockquote>
<p>输入： root = [1,2]</p>
<p>输出： 1</p>
</blockquote>
<hr>
<h4 id="提示"><a href="#提示">提示：</a></h4>
<ul>
<li>
<p>树中节点数目在范围 <code>[1, 10^4]</code> 内</p>
</li>
<li>
<p><code>-100 &#x3C;= Node.val &#x3C;= 100</code></p>
</li>
</ul>
<hr>
<h3 id="解答-javascript"><a href="#解答-javascript">解答 (JavaScript)</a></h3>
<h4 id="解题思路"><a href="#解题思路">解题思路</a></h4>
<p>这个问题的核心是找到树中任意两点间的最长路径。这条路径必然会有一个“拐点”或者说“最高点”，整条路径就是从这个“拐点”分别向其左、右子树延伸构成的。</p>
<p>我们的算法策略就是遍历树中的每一个节点，并假设它就是这个“拐点”，然后计算以它为“拐点”的路径长度，即 <strong>左子树深度 + 右子树深度</strong>。在所有节点上都计算一次，取其中的最大值即为整棵树的直径。</p>
<p>为了实现这一点，我们采用深度优先搜索（DFS）的后序遍历方式：</p>
<ol>
<li>
<p>定义一个全局变量 <code>maxDiameter</code> 来记录和更新目前找到的最大直径。</p>
</li>
<li>
<p>设计一个递归函数 <code>dfs(node)</code>，它的核心任务有两个：</p>
<ul>
<li>
<p><strong>计算和更新直径</strong>：在当前节点，利用其左右子树的深度（通过递归获得）计算穿过当前节点的路径长度 (<code>左深度 + 右深度</code>)，并用此值更新 <code>maxDiameter</code>。</p>
</li>
<li>
<p><strong>返回当前节点的深度</strong>：函数必须返回以当前节点为根的子树的深度 (<code>1 + Math.max(左深度, 右深度)</code>)。这个返回值是提供给其父节点，作为父节点计算的“原材料”。</p>
</li>
</ul>
</li>
</ol>
<p>通过这种方式，递归从下至上返回深度的同时，我们在每一层都完成了对直径的计算和更新，从而保证了最终结果的正确性。</p>
