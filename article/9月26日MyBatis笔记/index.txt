<p>AppTest.java</p>
<pre><code class="language-java">package com.zr;

import static org.junit.Assert.assertTrue;

import com.zr.dao.UserDao;
import com.zr.vo.User;
import org.apache.ibatis.io.Resources;
import org.apache.ibatis.session.SqlSession;
import org.apache.ibatis.session.SqlSessionFactory;
import org.apache.ibatis.session.SqlSessionFactoryBuilder;
import org.junit.Test;

import java.io.IOException;
import java.io.Reader;

/**
 * Unit test for simple App.
 */
public class AppTest 
{
    /**
     * Rigorous Test :-)
     */
    @Test
    public void test01() throws IOException {
        Reader reader = Resources.getResourceAsReader("mybatis-config.xml");
        //SqlSessionFactory对象封装了需要连接的数据源、表-vo-dao对应关系
        SqlSessionFactory factory = new SqlSessionFactoryBuilder().build(reader);
        //相当于jdbc的connection对象, openSession(false)代表手动事务, openSession(true)代表自动事务, 默认为false
        SqlSession sqlSession = factory.openSession(false);
        //获取dao接口的实现类，该实现类由mybatis动态(动态代理)生成
        UserDao userDao = sqlSession.getMapper(UserDao.class);
        //调用实现类的方法，注意：接口方法必须和对应的映射.xml文件中标签的id值一致
        User user = userDao.selectByPrimaryKey("admin");
        //提交事务 sqlSession.commit();
        //回滚事务 sqlSession.rollback();
        System.out.println(user);
        //关闭连接
        sqlSession.close();
        reader.close();
    }
}
</code></pre>

<h2>OGNL 表达式</h2>
<p>OGNL(Object Graphic Navigation Language, 对象图导航语言)<br>
<img src="9_26_01.PNG" alt="alt"></p>
<ol>
<li>方法参数是String 或基本数据类型(包装器类型)<br>
<code>select(String userName)     #{userName}</code></li>
<li>参数是JavaBean对象类型<br>
<code>add(User user)     #{属性名} #{userId} #{password}</code></li>
<li>参数是Map的时候<br>
<code>add(Map&#x3C;String, Object>)     #{key} #{userId}</code></li>
<li>多参数1<br>
<code>queryList(@Param("a")String userId, @Param("b")String userName)     #{a} #{b}</code></li>
<li>多参数2<br>
<code>list(@Param("u") User user, @Param("d")String deptId)     #{u.userId} #{u.userName}    #{d}</code></li>
</ol>
<p><strong>注意</strong> 在xml中，如果参数是字符串类型，如<code>#{userName}===>'张三'</code> (自动加单引号), 写 <code>${userName}====>张三</code> 没有单引号</p>
<h2>resultMap 和 resultType的区别</h2>
<p>二者不可共存</p>
<p><code>resultType</code>: 方法返回值是具体的类型,数据库表的【列名】映射到返回值上，如果返回值是JavaBean, 那么返回值映射关系为：列名-->属性名<br>
<code>resultMap</code>: 自定义的映射关系</p>
