4:T48ba,<h3 id="101-对称二叉树-symmetric-tree"><a href="#101-对称二叉树-symmetric-tree">101. 对称二叉树 (Symmetric Tree)</a></h3>
<hr>
<h4 id="题目描述"><a href="#题目描述"><strong>题目描述</strong></a></h4>
<p>给你一个二叉树的根节点 <code>root</code> ， 检查它是否轴对称。</p>
<hr>
<h4 id="示例说明"><a href="#示例说明"><strong>示例说明</strong></a></h4>
<p><strong>示例 1：</strong></p>
<ul>
<li>
<p><strong>输入：</strong> <code>root = [1,2,2,3,4,4,3]</code></p>
</li>
<li>
<p><strong>输出：</strong> <code>true</code></p>
</li>
<li>
<p><strong>结构展示：</strong></p>
<p>Plaintext</p>
<pre><code class="hljs language-markdown"><span class="hljs-code">      1
     / \
    2   2
   / \ / \
  3  4 4  3
</span></code></pre>
</li>
</ul>
<p><strong>示例 2：</strong></p>
<ul>
<li>
<p><strong>输入：</strong> <code>root = [1,2,2,null,3,null,3]</code></p>
</li>
<li>
<p><strong>输出：</strong> <code>false</code></p>
</li>
<li>
<p><strong>结构展示：</strong></p>
<p>Plaintext</p>
<pre><code class="hljs language-markdown"><span class="hljs-code">      1
     / \
    2   2
     \   \
      3   3
</span></code></pre>
</li>
</ul>
<hr>
<h4 id="提示条件"><a href="#提示条件"><strong>提示条件</strong></a></h4>
<ul>
<li>
<p>树中节点数目在范围 <code>[1, 1000]</code> 内。</p>
</li>
<li>
<p>节点值范围：<code>-100 &#x3C;= Node.val &#x3C;= 100</code>。</p>
</li>
</ul>
<hr>
<h4 id="进阶挑战"><a href="#进阶挑战"><strong>进阶挑战</strong></a></h4>
<p>你可以运用<strong>递归</strong>和<strong>迭代</strong>两种方法解决这个问题吗？</p>
<hr>
<p>判断一个二叉树是否轴对称，本质上是在考察<strong>镜像结构</strong>。一个二叉树如果是对称的，那么它的左子树和右子树必须互为镜像。</p>
<h3 id="1-核心思路第一性原理拆解"><a href="#1-核心思路第一性原理拆解">1. 核心思路：第一性原理拆解</a></h3>
<p>我们要判断两棵树（假设为 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>L</mi></mrow><annotation encoding="application/x-tex">L</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal">L</span></span></span></span> 和 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>R</mi></mrow><annotation encoding="application/x-tex">R</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal" style="margin-right:0.00773em;">R</span></span></span></span>）是否互为镜像，必须同时满足以下三个条件：</p>
<ol>
<li>
<p><strong>根节点值相等</strong>：<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>L</mi><mi mathvariant="normal">.</mi><mi>v</mi><mi>a</mi><mi>l</mi><mo>=</mo><mi>R</mi><mi mathvariant="normal">.</mi><mi>v</mi><mi>a</mi><mi>l</mi></mrow><annotation encoding="application/x-tex">L.val = R.val</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6944em;"></span><span class="mord mathnormal">L</span><span class="mord">.</span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span><span class="mord mathnormal">a</span><span class="mord mathnormal" style="margin-right:0.01968em;">l</span><span class="mspace" style="margin-right:0.2778em;"></span><span class="mrel">=</span><span class="mspace" style="margin-right:0.2778em;"></span></span><span class="base"><span class="strut" style="height:0.6944em;"></span><span class="mord mathnormal" style="margin-right:0.00773em;">R</span><span class="mord">.</span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span><span class="mord mathnormal">a</span><span class="mord mathnormal" style="margin-right:0.01968em;">l</span></span></span></span>。</p>
</li>
<li>
<p><strong>外侧对称</strong>：<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>L</mi></mrow><annotation encoding="application/x-tex">L</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal">L</span></span></span></span> 的左子树与 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>R</mi></mrow><annotation encoding="application/x-tex">R</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal" style="margin-right:0.00773em;">R</span></span></span></span> 的右子树互为镜像。</p>
</li>
<li>
<p><strong>内侧对称</strong>：<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>L</mi></mrow><annotation encoding="application/x-tex">L</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal">L</span></span></span></span> 的右子树与 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>R</mi></mrow><annotation encoding="application/x-tex">R</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.6833em;"></span><span class="mord mathnormal" style="margin-right:0.00773em;">R</span></span></span></span> 的左子树互为镜像。</p>
</li>
</ol>
<p>这是一种典型的递归定义。如果根节点为空，它显然是对称的。如果根节点存在，我们就开始比较它的左孩子和右孩子。</p>
<hr>
<h3 id="2-递归实现dfs"><a href="#2-递归实现dfs">2. 递归实现（DFS）</a></h3>
<p>这是最直观的解法。我们定义一个辅助函数 <code>isMirror</code> 来处理两个节点的比较。</p>
<h4 id="代码实现"><a href="#代码实现">代码实现：</a></h4>
<p>JavaScript</p>
<pre><code class="hljs language-kotlin"><span class="hljs-comment">/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 * this.val = (val===undefined ? 0 : val)
 * this.left = (left===undefined ? null : left)
 * this.right = (right===undefined ? null : right)
 * }
 */</span>
<span class="hljs-comment">/**
 * <span class="hljs-doctag">@param</span> {TreeNode} root
 * <span class="hljs-doctag">@return</span> {boolean}
 */</span>
<span class="hljs-keyword">var</span> isSymmetric = function(root) {
    <span class="hljs-keyword">if</span> (!root) <span class="hljs-keyword">return</span> <span class="hljs-literal">true</span>;

    <span class="hljs-keyword">const</span> isMirror = (l, r) => {
        <span class="hljs-comment">// 1. 同时为空，对称</span>
        <span class="hljs-keyword">if</span> (!l &#x26;&#x26; !r) <span class="hljs-keyword">return</span> <span class="hljs-literal">true</span>;
        <span class="hljs-comment">// 2. 一个为空或值不相等，不对称</span>
        <span class="hljs-keyword">if</span> (!l || !r || l.<span class="hljs-keyword">val</span> !== r.<span class="hljs-keyword">val</span>) <span class="hljs-keyword">return</span> <span class="hljs-literal">false</span>;
        
        <span class="hljs-comment">// 3. 递归比较：左的左 vs 右的右（外侧） AND 左的右 vs 右的左（内侧）</span>
        <span class="hljs-keyword">return</span> isMirror(l.left, r.right) &#x26;&#x26; isMirror(l.right, r.left);
    };

    <span class="hljs-keyword">return</span> isMirror(root.left, root.right);
};
</code></pre>
<hr>
<h3 id="3-迭代实现bfs队列"><a href="#3-迭代实现bfs队列">3. 迭代实现（BFS/队列）</a></h3>
<p>如果你担心树的深度过大导致递归栈溢出，或者面试官要求非递归解法，可以使用队列（Queue）来模拟递归的过程。其逻辑是：<strong>成对地将需要比较的节点放入队列</strong>。</p>
<h4 id="思路"><a href="#思路">思路：</a></h4>
<ol>
<li>
<p>初始化一个队列，将 <code>root.left</code> 和 <code>root.right</code> 入队。</p>
</li>
<li>
<p>每次从队列中取出两个节点 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>u</mi></mrow><annotation encoding="application/x-tex">u</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal">u</span></span></span></span> 和 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>v</mi></mrow><annotation encoding="application/x-tex">v</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span></span></span></span>。</p>
</li>
<li>
<p>判断 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>u</mi></mrow><annotation encoding="application/x-tex">u</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal">u</span></span></span></span> 和 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>v</mi></mrow><annotation encoding="application/x-tex">v</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span></span></span></span> 是否满足镜像条件：</p>
<ul>
<li>
<p>都为空：继续下一轮循环。</p>
</li>
<li>
<p>其中一个为空或值不等：返回 <code>false</code>。</p>
</li>
</ul>
</li>
<li>
<p>将 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>u</mi></mrow><annotation encoding="application/x-tex">u</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal">u</span></span></span></span> 的左孩子和 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>v</mi></mrow><annotation encoding="application/x-tex">v</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span></span></span></span> 的右孩子成对入队（外侧比较）。</p>
</li>
<li>
<p>将 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>u</mi></mrow><annotation encoding="application/x-tex">u</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal">u</span></span></span></span> 的右孩子和 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>v</mi></mrow><annotation encoding="application/x-tex">v</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.4306em;"></span><span class="mord mathnormal" style="margin-right:0.03588em;">v</span></span></span></span> 的左孩子成对入队（内侧比较）。</p>
</li>
</ol>
<h4 id="代码实现-1"><a href="#代码实现-1">代码实现：</a></h4>
<p>JavaScript</p>
<pre><code class="hljs language-kotlin"><span class="hljs-keyword">var</span> isSymmetric = function(root) {
    <span class="hljs-keyword">if</span> (!root) <span class="hljs-keyword">return</span> <span class="hljs-literal">true</span>;
    
    <span class="hljs-comment">// 使用队列存储成对比较的节点</span>
    let queue = [root.left, root.right];
    
    <span class="hljs-keyword">while</span> (queue.length > <span class="hljs-number">0</span>) {
        let l = queue.shift();
        let r = queue.shift();
        
        <span class="hljs-keyword">if</span> (!l &#x26;&#x26; !r) <span class="hljs-keyword">continue</span>;
        <span class="hljs-keyword">if</span> (!l || !r || l.<span class="hljs-keyword">val</span> !== r.<span class="hljs-keyword">val</span>) <span class="hljs-keyword">return</span> <span class="hljs-literal">false</span>;
        
        <span class="hljs-comment">// 注意入队顺序，确保下次取出的两个是镜像位置的节点</span>
        queue.push(l.left, r.right); <span class="hljs-comment">// 外侧</span>
        queue.push(l.right, r.left); <span class="hljs-comment">// 内侧</span>
    }
    
    <span class="hljs-keyword">return</span> <span class="hljs-literal">true</span>;
};
</code></pre>

<ul>
<li>
