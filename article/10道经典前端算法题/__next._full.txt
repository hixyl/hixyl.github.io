<h4 id="1-防抖debounce与节流throttle"><a href="#1-防抖debounce与节流throttle">1. 防抖（Debounce）与节流（Throttle）</a></h4>
<p><strong>考察重点：</strong> 闭包、高阶函数、时间机制（宏任务）。</p>
<p><strong>解析：</strong> * <strong>防抖</strong>就像是“电梯门机制”：只要一直有人进出，电梯门就不会关。只有等最后一个人进入后，且一段时间内没人再来，门才会关上并运行。用于搜索框联想。</p>
<ul>
<li><strong>节流</strong>就像是“水龙头滴水”：无论你把水龙头拧多大，水滴总是每隔固定的时间掉下来一滴。用于滚动事件监听、按钮防连击。</li>
</ul>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">// 防抖实现</span>

<h4 id="2-带有循环引用检测的深拷贝-deep-clone"><a href="#2-带有循环引用检测的深拷贝-deep-clone">2. 带有循环引用检测的深拷贝 (Deep Clone)</a></h4>
<p><strong>考察重点：</strong> 数据类型判断、递归、内存引用（使用 <code>WeakMap</code> 解决循环引用避免内存泄漏）。</p>
<p><strong>解析：</strong> 浅拷贝只是复制了房间的钥匙，深拷贝是造一栋完全一样的房子。难点在于，如果房子 A 的卧室里挂着一幅画，画里画着房子 A 本身（循环引用），普通的递归会陷入死循环。我们需要一个“记事本”（<code>WeakMap</code>）来记录“哪些房子已经建过了”，遇到建过的直接拿现成的。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">deepClone</span>(<span class="hljs-params">obj, hash = <span class="hljs-keyword">new</span> <span class="hljs-built_in">WeakMap</span>()</span>) {
    <span class="hljs-comment">// 1. 处理基本类型和 null</span>
    <span class="hljs-keyword">if</span> (obj === <span class="hljs-literal">null</span> || <span class="hljs-keyword">typeof</span> obj !== <span class="hljs-string">'object'</span>) <span class="hljs-keyword">return</span> obj;
    <span class="hljs-comment">// 2. 处理特殊对象（日期、正则）</span>
    <span class="hljs-keyword">if</span> (obj <span class="hljs-keyword">instanceof</span> <span class="hljs-title class_">Date</span>) <span class="hljs-keyword">return</span> <span class="hljs-keyword">new</span> <span class="hljs-title class_">Date</span>(obj);
    <span class="hljs-keyword">if</span> (obj <span class="hljs-keyword">instanceof</span> <span class="hljs-title class_">RegExp</span>) <span class="hljs-keyword">return</span> <span class="hljs-keyword">new</span> <span class="hljs-title class_">RegExp</span>(obj);

    <span class="hljs-comment">// 3. 查“记事本”，如果这个对象之前拷贝过，直接返回其克隆体，防止死循环</span>
    <span class="hljs-keyword">if</span> (hash.<span class="hljs-title function_">has</span>(obj)) <span class="hljs-keyword">return</span> hash.<span class="hljs-title function_">get</span>(obj);

    <span class="hljs-comment">// 4. 创建新对象/数组（保持原型链）</span>
    <span class="hljs-keyword">const</span> cloneObj = <span class="hljs-keyword">new</span> obj.<span class="hljs-title function_">constructor</span>(<span class="hljs-params"></span>);
    hash.<span class="hljs-title function_">set</span>(obj, cloneObj); <span class="hljs-comment">// 记下：这个旧对象对应的克隆体是谁</span>

    <span class="hljs-comment">// 5. 递归拷贝属性</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> key <span class="hljs-keyword">in</span> obj) {
        <span class="hljs-keyword">if</span> (obj.<span class="hljs-title function_">hasOwnProperty</span>(key)) {
            cloneObj[key] = <span class="hljs-title function_">deepClone</span>(obj[key], hash);
        }
    }
    <span class="hljs-keyword">return</span> cloneObj;
}

<h4 id="3-手写-promiseall"><a href="#3-手写-promiseall">3. 手写 Promise.all</a></h4>
<p><strong>考察重点：</strong> Promise 规范、异步并发控制、状态流转。</p>
<p><strong>解析：</strong> 想象你在组装一台电脑，你需要等待 CPU、主板、显卡（各个 Promise）都到货才能开始组装（<code>resolve</code>），但只要有一件快递在中途丢失或损坏（<code>reject</code>），整个装机计划就宣告失败。注意，最终零件的摆放顺序必须和购买清单的顺序一致，而不能是谁先到货谁排前面。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">myPromiseAll</span>(<span class="hljs-params">promises</span>) {
    <span class="hljs-keyword">return</span> <span class="hljs-keyword">new</span> <span class="hljs-title class_">Promise</span>(<span class="hljs-function">(<span class="hljs-params">resolve, reject</span>) =></span> {
        <span class="hljs-comment">// 处理非数组或空迭代器的情况</span>
        <span class="hljs-keyword">if</span> (!<span class="hljs-title class_">Array</span>.<span class="hljs-title function_">isArray</span>(promises)) {
            <span class="hljs-keyword">return</span> <span class="hljs-title function_">reject</span>(<span class="hljs-keyword">new</span> <span class="hljs-title class_">TypeError</span>(<span class="hljs-string">'Argument must be an iterable'</span>));
        }
        <span class="hljs-keyword">if</span> (promises.<span class="hljs-property">length</span> === <span class="hljs-number">0</span>) <span class="hljs-keyword">return</span> <span class="hljs-title function_">resolve</span>([]);

        <span class="hljs-keyword">let</span> resolvedCount = <span class="hljs-number">0</span>;
        <span class="hljs-keyword">const</span> result = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Array</span>(promises.<span class="hljs-property">length</span>); <span class="hljs-comment">// 预留固定长度的数组保证顺序</span>

        promises.<span class="hljs-title function_">forEach</span>(<span class="hljs-function">(<span class="hljs-params">p, index</span>) =></span> {
            <span class="hljs-comment">// 使用 Promise.resolve 包装，防止传入的不是 Promise 对象</span>
            <span class="hljs-title class_">Promise</span>.<span class="hljs-title function_">resolve</span>(p).<span class="hljs-title function_">then</span>(
                <span class="hljs-function"><span class="hljs-params">val</span> =></span> {
                    result[index] = val; <span class="hljs-comment">// 按原索引存入结果，而非按完成时间</span>
                    resolvedCount++;
                    <span class="hljs-comment">// 所有任务都成功完成时，整体 resolve</span>
                    <span class="hljs-keyword">if</span> (resolvedCount === promises.<span class="hljs-property">length</span>) {
                        <span class="hljs-title function_">resolve</span>(result);
                    }
                },
                <span class="hljs-function"><span class="hljs-params">err</span> =></span> {
                    <span class="hljs-comment">// 只要有一个失败，整体立即失败</span>
                    <span class="hljs-title function_">reject</span>(err);
                }
            );
        });
    });
}

<h4 id="4-发布订阅模式-eventbus"><a href="#4-发布订阅模式-eventbus">4. 发布订阅模式 (EventBus)</a></h4>
<p><strong>考察重点：</strong> 设计模式、组件间解耦通信机制。</p>
<p><strong>解析：</strong> 就像是 B 站的 UP 主（发布者）和粉丝（订阅者）。粉丝在 B 站（EventBus）点了关注（<code>on</code>），UP 主发视频时告诉 B 站（<code>emit</code>），B 站负责把通知推送给所有粉丝。取消关注就是 <code>off</code>，只看一次就是 <code>once</code>。</p>
<p>代码段</p>
<pre><code class="hljs language-mermaid">graph LR
    Sub1[订阅者 A] -->|"(1) on 'update'"| Bus(EventBus 调度中心)
    Sub2[订阅者 B] -->|"(1) on 'update'"| Bus
    Pub[发布者] -->|"(2) emit 'update', data"| Bus
    Bus -->|"(3) 执行回调传参"| Sub1
    Bus -->|"(3) 执行回调传参"| Sub2
</code></pre>

<h3 id="第二部分业务数据结构转换"><a href="#第二部分业务数据结构转换">第二部分：业务数据结构转换</a></h3>
<p>日常前端开发重度依赖数据处理，尤其是在渲染菜单、权限树、表格展开行时。</p>
<h4 id="5-扁平数组转树形结构-array-to-tree"><a href="#5-扁平数组转树形结构-array-to-tree">5. 扁平数组转树形结构 (Array to Tree)</a></h4>
<p><strong>考察重点：</strong> 引用类型特性、空间换时间（构建哈希表降低时间复杂度至 $O(N)$）。</p>
<p><strong>解析：</strong> 给定一堆打乱的人事档案卡片，上面写着自己的 ID 和直属上司的 ID（<code>parentId</code>）。常规做法是拿一张卡片去扫遍所有人找上司（复杂度 $O(N^2)$）。高效做法是：先按 ID 给所有人建一个目录字典，然后遍历一次，每个人只需翻一次字典找到上司，把自己挂到上司的 <code>children</code> 列表里。因为 JavaScript 中对象是按引用传递的，挂在字典里的变动会自动反映到最终的树中。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">// 输入示例：[{id: 1, pid: 0}, {id: 2, pid: 1}, {id: 3, pid: 1}]</span>
<span class="hljs-keyword">function</span> <span class="hljs-title function_">arrayToTree</span>(<span class="hljs-params">items</span>) {
    <span class="hljs-keyword">const</span> result = [];   <span class="hljs-comment">// 存放根节点</span>
    <span class="hljs-keyword">const</span> itemMap = {};  <span class="hljs-comment">// 字典</span>

    <span class="hljs-comment">// 1. 先把所有项放入字典，并初始化 children 数组</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">const</span> item <span class="hljs-keyword">of</span> items) {
        itemMap[item.<span class="hljs-property">id</span>] = { ...item, <span class="hljs-attr">children</span>: [] };
    }

    <span class="hljs-comment">// 2. 遍历字典构建树</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">const</span> item <span class="hljs-keyword">of</span> items) {
        <span class="hljs-keyword">const</span> id = item.<span class="hljs-property">id</span>;
        <span class="hljs-keyword">const</span> pid = item.<span class="hljs-property">pid</span>;
        <span class="hljs-keyword">const</span> treeItem = itemMap[id];
        
        <span class="hljs-keyword">if</span> (pid === <span class="hljs-number">0</span>) {
            <span class="hljs-comment">// pid 为 0 代表是根节点，直接放入结果集</span>
            result.<span class="hljs-title function_">push</span>(treeItem);
        } <span class="hljs-keyword">else</span> {
            <span class="hljs-comment">// 否则，找到其父节点，将自己推入父节点的 children 中</span>
            <span class="hljs-keyword">if</span> (!itemMap[pid]) {
                itemMap[pid] = { <span class="hljs-attr">children</span>: [] };
            }
            itemMap[pid].<span class="hljs-property">children</span>.<span class="hljs-title function_">push</span>(treeItem);
        }
    }
    <span class="hljs-keyword">return</span> result;
}
</code></pre>
<h4 id="6-版本号排序"><a href="#6-版本号排序">6. 版本号排序</a></h4>
<p><strong>考察重点：</strong> 字符串处理、边界条件思考（长度不一的处理）。</p>
<p><strong>解析：</strong> 前端经常需要判断 App 版本（如 <code>1.10.2</code> > <code>1.2.3</code>）。不能直接按字符串比，因为按字典序 <code>'10'</code> 会排在 <code>'2'</code> 前面。我们需要把它们根据 <code>.</code> 切割成数字数组，像比较年、月、日一样逐个身位进行较量。缺位的地方自动补零（比如 <code>1.2</code> 和 <code>1.2.0</code> 相等）。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">// arr = ['0.1.1', '2.3.3', '0.302.1', '4.2', '4.3.5', '4.3.4.5']</span>
<span class="hljs-keyword">function</span> <span class="hljs-title function_">sortVersions</span>(<span class="hljs-params">versions</span>) {
    <span class="hljs-keyword">return</span> versions.<span class="hljs-title function_">sort</span>(<span class="hljs-function">(<span class="hljs-params">a, b</span>) =></span> {
        <span class="hljs-keyword">const</span> v1 = a.<span class="hljs-title function_">split</span>(<span class="hljs-string">'.'</span>);
        <span class="hljs-keyword">const</span> v2 = b.<span class="hljs-title function_">split</span>(<span class="hljs-string">'.'</span>);
        <span class="hljs-keyword">const</span> maxLen = <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">max</span>(v1.<span class="hljs-property">length</span>, v2.<span class="hljs-property">length</span>);

        <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> i = <span class="hljs-number">0</span>; i &#x3C; maxLen; i++) {
            <span class="hljs-comment">// 如果某一位没有，默认为 0</span>
            <span class="hljs-keyword">const</span> num1 = <span class="hljs-built_in">parseInt</span>(v1[i] || <span class="hljs-number">0</span>); 
            <span class="hljs-keyword">const</span> num2 = <span class="hljs-built_in">parseInt</span>(v2[i] || <span class="hljs-number">0</span>);

            <span class="hljs-keyword">if</span> (num1 > num2) <span class="hljs-keyword">return</span> <span class="hljs-number">1</span>;  <span class="hljs-comment">// a > b，a 排后面</span>
            <span class="hljs-keyword">if</span> (num1 &#x3C; num2) <span class="hljs-keyword">return</span> -<span class="hljs-number">1</span>; <span class="hljs-comment">// a &#x3C; b，a 排前面</span>
            <span class="hljs-comment">// 如果相等，继续循环看下一位</span>
        }
        <span class="hljs-keyword">return</span> <span class="hljs-number">0</span>; <span class="hljs-comment">// 完全相等</span>
    });
}

<h3 id="第三部分经典算法前端高频"><a href="#第三部分经典算法前端高频">第三部分：经典算法（前端高频）</a></h3>
<p>前端的算法题往往侧重于 DOM 解析（树、栈）、缓存（哈希表、链表）以及字符处理。</p>
<h4 id="7-有效的括号-valid-parentheses"><a href="#7-有效的括号-valid-parentheses">7. 有效的括号 (Valid Parentheses)</a></h4>
<p><strong>考察重点：</strong> 栈（Stack）的后进先出（LIFO）特性。</p>
<p><strong>解析：</strong> 浏览器解析 HTML 标签时就是用的类似原理（遇到 <code>&#x3C;div></code> 压栈，遇到 <code>&#x3C;/div></code> 出栈）。规则很简单：遇到左括号就去“等待室”（栈）排队；遇到右括号，看看等待室最靠门的那个人是不是跟自己配对的。如果配对，那人离开；如果不配对或者等待室没人了，说明格式报错。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">isValid</span>(<span class="hljs-params">s</span>) {
    <span class="hljs-comment">// 奇数长度必然不匹配</span>
    <span class="hljs-keyword">if</span> (s.<span class="hljs-property">length</span> % <span class="hljs-number">2</span> !== <span class="hljs-number">0</span>) <span class="hljs-keyword">return</span> <span class="hljs-literal">false</span>;
    
    <span class="hljs-keyword">const</span> stack = [];
    <span class="hljs-comment">// 用 Map 存储配对关系，右括号为键，左括号为值</span>
    <span class="hljs-keyword">const</span> map = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Map</span>([
        [<span class="hljs-string">')'</span>, <span class="hljs-string">'('</span>],
        [<span class="hljs-string">']'</span>, <span class="hljs-string">'['</span>],
        [<span class="hljs-string">'}'</span>, <span class="hljs-string">'{'</span>]
    ]);

    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> char <span class="hljs-keyword">of</span> s) {
        <span class="hljs-keyword">if</span> (map.<span class="hljs-title function_">has</span>(char)) {
            <span class="hljs-comment">// 如果是右括号</span>
            <span class="hljs-comment">// 弹出栈顶元素，如果栈为空则弹出一个伪造值 '#'</span>
            <span class="hljs-keyword">const</span> topElement = stack.<span class="hljs-property">length</span> ? stack.<span class="hljs-title function_">pop</span>() : <span class="hljs-string">'#'</span>;
            <span class="hljs-keyword">if</span> (topElement !== map.<span class="hljs-title function_">get</span>(char)) {
                <span class="hljs-keyword">return</span> <span class="hljs-literal">false</span>; <span class="hljs-comment">// 栈顶元素与当前右括号不匹配</span>
            }
        } <span class="hljs-keyword">else</span> {
            <span class="hljs-comment">// 如果是左括号，压入栈</span>
            stack.<span class="hljs-title function_">push</span>(char);
        }
    }
    <span class="hljs-comment">// 最后检查栈是否被清空，空了说明全匹配</span>
    <span class="hljs-keyword">return</span> stack.<span class="hljs-property">length</span> === <span class="hljs-number">0</span>;
}

<h4 id="8-lru-缓存策略-least-recently-used"><a href="#8-lru-缓存策略-least-recently-used">8. LRU 缓存策略 (Least Recently Used)</a></h4>
<p><strong>考察重点：</strong> 哈希表与双向链表的结合。</p>
<p><strong>解析：</strong> 想象你的衣柜空间有限（<code>capacity</code>）。你每次买新衣服（<code>put</code>）都挂在最显眼的位置；你每次穿某件旧衣服（<code>get</code>），也会把它重新挂到最显眼的位置。当衣柜满了，再买新衣服时，只能把挂在最角落（最久没被碰过）的那件扔掉。在 JavaScript 中，原生的 <code>Map</code> 对象非常神奇，它<strong>维护了键值对的插入顺序</strong>，我们恰好利用这个特性，避开手动写繁琐的双向链表。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-keyword">class</span> <span class="hljs-title class_">LRUCache</span> {
    <span class="hljs-title function_">constructor</span>(<span class="hljs-params">capacity</span>) {
        <span class="hljs-variable language_">this</span>.<span class="hljs-property">capacity</span> = capacity;
        <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span> = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Map</span>(); <span class="hljs-comment">// Map 既能 O(1) 查找，又能记录插入顺序</span>
    }

    <span class="hljs-title function_">get</span>(<span class="hljs-params">key</span>) {
        <span class="hljs-keyword">if</span> (!<span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">has</span>(key)) <span class="hljs-keyword">return</span> -<span class="hljs-number">1</span>;
        <span class="hljs-comment">// 核心：如果有访问，把它从原位置删掉，重新插入，使其变成“最新鲜”的</span>
        <span class="hljs-keyword">const</span> val = <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">get</span>(key);
        <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">delete</span>(key);
        <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">set</span>(key, val);
        <span class="hljs-keyword">return</span> val;
    }

    <span class="hljs-title function_">put</span>(<span class="hljs-params">key, value</span>) {
        <span class="hljs-comment">// 如果已存在，先删除，再重新 set，更新新鲜度</span>
        <span class="hljs-keyword">if</span> (<span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">has</span>(key)) {
            <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">delete</span>(key);
        }
        <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">set</span>(key, value);

        <span class="hljs-comment">// 如果超出容量限制，删除最老的元素</span>
        <span class="hljs-keyword">if</span> (<span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-property">size</span> > <span class="hljs-variable language_">this</span>.<span class="hljs-property">capacity</span>) {
            <span class="hljs-comment">// map.keys().next().value 能获取到 Map 中最先插入的键（最老的）</span>
            <span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">delete</span>(<span class="hljs-variable language_">this</span>.<span class="hljs-property">cache</span>.<span class="hljs-title function_">keys</span>().<span class="hljs-title function_">next</span>().<span class="hljs-property">value</span>);

<h4 id="9-无重复字符的最长子串"><a href="#9-无重复字符的最长子串">9. 无重复字符的最长子串</a></h4>
<p><strong>考察重点：</strong> 滑动窗口（双指针）、哈希集合（Set）。</p>
<p><strong>解析：</strong> 就像一条贪吃蛇在字符串上爬。蛇头（右指针）不断往前探，只要没遇到吃过的字母（不在 Set 里），就吞进去变长。如果碰到了吃过的字母，蛇尾（左指针）就开始往外拉屎排空，直到把那个重复的字母排掉为止。记录下这条蛇达到过的最大长度。</p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">lengthOfLongestSubstring</span>(<span class="hljs-params">s</span>) {

        <span class="hljs-comment">// 如果遇到重复字符，左边界开始向右收缩，直到移出重复字符</span>
        <span class="hljs-keyword">while</span> (set.<span class="hljs-title function_">has</span>(s[right])) {
            set.<span class="hljs-title function_">delete</span>(s[left]);
            left++;
        }
        <span class="hljs-comment">// 右边界对应的新字符加入窗口</span>
        set.<span class="hljs-title function_">add</span>(s[right]);
        <span class="hljs-comment">// 更新最大长度记录</span>
        maxLength = <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">max</span>(maxLength, right - left + <span class="hljs-number">1</span>);
    }

    <span class="hljs-keyword">return</span> maxLength;
}

<h4 id="10-二叉树的层序遍历-level-order-traversal"><a href="#10-二叉树的层序遍历-level-order-traversal">10. 二叉树的层序遍历 (Level Order Traversal)</a></h4>
