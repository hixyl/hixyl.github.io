4:T1d82,<h3 id="leetcode-53-最大子数组和"><a href="#leetcode-53-最大子数组和"><strong>LeetCode 53. 最大子数组和</strong></a></h3>
<p><strong>难度：中等</strong></p>
<h4 id="题目描述"><a href="#题目描述"><strong>题目描述</strong></a></h4>
<p>给你一个整数数组 <code>nums</code> ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。</p>
<p><strong>子数组</strong> 是数组中的一个连续部分。</p>
<h4 id="示例"><a href="#示例"><strong>示例</strong></a></h4>
<blockquote>
<p>示例 1：</p>
<p>输入：nums = [-2,1,-3,4,-1,2,1,-5,4]</p>
<p>输出：6</p>
<p>**解释：**连续子数组 [4,-1,2,1] 的和最大，为 6 。</p>
</blockquote>
<blockquote>
<p>示例 2：</p>
<p>输入：nums = [1]</p>
<p>输出：1</p>
</blockquote>
<blockquote>
<p>示例 3：</p>
<p>输入：nums = [5,4,-1,7,8]</p>
<p>输出：23</p>
</blockquote>
<h4 id="提示"><a href="#提示"><strong>提示</strong></a></h4>
<ul>
<li>
<p>1lenums.lengthle105</p>
</li>
<li>
<p>−104lenums[i]le104</p>
</li>
</ul>
<hr>
<h3 id="解题思路与代码实现"><a href="#解题思路与代码实现"><strong>解题思路与代码实现</strong></a></h3>
<h4 id="核心思想"><a href="#核心思想"><strong>核心思想</strong></a></h4>
<p>这道题是动态规划中的一个经典问题，最优雅的解法被称为“卡丹算法”（Kadane's Algorithm）。其核心思想是，当我们从左到右遍历数组时，对于每一个元素，我们都做出一个决策：是<strong>将当前元素加入到前面的子数组以延续它</strong>，还是<strong>从当前元素开始一个新的子数组</strong>。</p>
<p>决策的依据非常简单：如果前一个子数组的和为负数，那么它对于后续的求和只会起到“拖累”的作用。在这种情况下，我们不如抛弃它，直接从当前元素开始计算一个新的子数组和。</p>
<p>基于此，我们只需要维护两个变量：</p>
<ol>
<li>
<p><code>currentMax</code>：记录以<strong>当前元素</strong>为结尾的连续子数组的最大和。</p>
</li>
<li>
<p><code>globalMax</code>：记录在整个遍历过程中所遇到的<strong>全局</strong>最大子数组和。</p>
</li>
</ol>
<p>遍历完整个数组，<code>globalMax</code> 中存储的就是最终答案。</p>
<h4 id="javascript-代码实现"><a href="#javascript-代码实现"><strong>JavaScript 代码实现</strong></a></h4>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">/**
 * <span class="hljs-doctag">@param</span> {<span class="hljs-type">number[]</span>} <span class="hljs-variable">nums</span>
 * <span class="hljs-doctag">@return</span> {<span class="hljs-type">number</span>}
 */</span>
<span class="hljs-keyword">const</span> <span class="hljs-title function_">maxSubArray</span> = (<span class="hljs-params">nums</span>) => {
    <span class="hljs-comment">// globalMax 用于存储全局的最大子数组和。</span>
    <span class="hljs-comment">// 根据题目约束，子数组至少包含一个元素，因此可以安全地初始化为数组的第一个元素。</span>
    <span class="hljs-keyword">let</span> globalMax = nums[<span class="hljs-number">0</span>];
    
    <span class="hljs-comment">// currentMax 用于存储以当前元素结尾的连续子数组的最大和。</span>
    <span class="hljs-keyword">let</span> currentMax = nums[<span class="hljs-number">0</span>];

    <span class="hljs-comment">// 从第二个元素开始遍历数组</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> i = <span class="hljs-number">1</span>; i &#x3C; nums.<span class="hljs-property">length</span>; i++) {
        <span class="hljs-keyword">const</span> num = nums[i];
        
        <span class="hljs-comment">// 核心决策：</span>
        <span class="hljs-comment">// 是将当前数字 num 加入之前的子数组（currentMax + num），</span>
        <span class="hljs-comment">// 还是从 num 本身开始一个新的子数组？</span>
        <span class="hljs-comment">// 我们选择两者中和更大的那个作为新的 currentMax。</span>
        currentMax = <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">max</span>(num, currentMax + num);
        
        <span class="hljs-comment">// 在更新了以当前元素结尾的最大和之后，</span>
        <span class="hljs-comment">// 我们需要检查它是否也超过了已知的全局最大和。</span>
        <span class="hljs-keyword">if</span> (currentMax > globalMax) {
            globalMax = currentMax;
        }
    }

    <span class="hljs-keyword">return</span> globalMax;
};
</code></pre>
