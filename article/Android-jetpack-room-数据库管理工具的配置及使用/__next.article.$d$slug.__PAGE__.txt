4:T2900,<h2 id="配置"><a href="#配置">配置</a></h2>
<p>在根目录的<code>build.gradle</code>的<code>plugin</code>下添加</p>
<pre><code class="hljs language-gradle">id 'com.google.devtools.ksp' version '1.8.21-1.0.11' apply false
</code></pre>
<p>注意，版本号<code>1.8.21</code>要与<code>org.jetbrains.kotlin.android</code>版本号一致。示例:</p>
<p><code>/build.gradle</code></p>
<pre><code class="hljs language-gradle">plugins {
    id 'com.android.application' version '8.0.1' apply false
    id 'com.android.library' version '8.0.1' apply false
    id 'org.jetbrains.kotlin.android' version '1.8.21' apply false
    id 'com.google.devtools.ksp' version '1.8.21-1.0.11' apply false
}
</code></pre>
<p>在<code>app/build.gradle</code>下添加</p>
<pre><code class="hljs language-gradle"> def room_version = "2.5.0"

    implementation "androidx.room:room-runtime:$room_version"
    annotationProcessor "androidx.room:room-compiler:$room_version"

    // To use Kotlin Symbol Processing (KSP)
    ksp "androidx.room:room-compiler:$room_version"


    // optional - Test helpers
    testImplementation "androidx.room:room-testing:$room_version"
</code></pre>
<p>将jvm版本改为17</p>
<pre><code class="hljs language-gradle"> compileOptions {
        sourceCompatibility JavaVersion.VERSION_17
        targetCompatibility JavaVersion.VERSION_17
    }
    kotlinOptions {
        jvmTarget = '17'
    }
</code></pre>
