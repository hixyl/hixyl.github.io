4:T3af8,<p><strong>难度：中等</strong></p>
<p>给你一个整数数组 <code>nums</code> ，找到其中最长严格递增子序列的长度。1</p>
<p><strong>子序列</strong> 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，<code>[3,6,2,7]</code> 是数组 <code>[0,3,1,6,2,2,7]</code> 的子序列。2</p>
<hr>
<p>示例 1：</p>
<p>输入： nums = [10,9,2,5,3,7,101,18]</p>
<p>输出： 4</p>
<p>解释： 最长递增子序列是 [2,3,7,101]，因此长度为 4 。</p>
<p>示例 2：</p>
<p>输入： nums = [0,1,0,3,2,3]</p>
<p>输出： 4</p>
<p>示例 3：</p>
<p>输入： nums = [7,7,7,7,7,7,7]</p>
<p>输出： 1</p>
<hr>
<p><strong>提示：10</strong></p>
<ul>
<li>
<p><code>1 &#x3C;= nums.length &#x3C;= 2500</code></p>
</li>
<li>
<p><code>-10^4 &#x3C;= nums[i] &#x3C;= 10^4</code></p>
</li>
</ul>
<hr>
<p>进阶：</p>
<p>你能将算法的时间复杂度降低到 O(nlogn) 吗?</p>
<hr>
<h3 id="解答"><a href="#解答">解答</a></h3>
<p>解决这个问题有两种主流方法：一种是基础的动态规划，另一种是更高效的、结合了贪心和二分查找的优化方法。这里将提供完整的两种解法。</p>
<p><strong>最优解是第二种方法</strong>，它满足了题目进阶部分的要求。</p>
<h3 id="解法一贪心--二分查找-最优解"><a href="#解法一贪心--二分查找-最优解">解法一：贪心 + 二分查找 (最优解)</a></h3>
<p>这种方法是本题的进阶解法，时间复杂度为 O(nlogn)。</p>
<p>核心思想：</p>
<p>我们维护一个 tails 数组，tails[i] 存储的是所有长度为 i+1 的递增子序列中，结尾数字最小的那个值。这个 tails 数组一定是单调递增的。</p>
<p>我们遍历原始数组 <code>nums</code> 中的每个数字 <code>num</code>：</p>
<ol>
<li>
<p>如果 <code>num</code> 比 <code>tails</code> 数组的所有元素都大，就意味着 <code>num</code> 可以接在当前最长的子序列之后，形成一个更长的子序列。我们直接将其追加到 <code>tails</code> 数组的末尾。</p>
</li>
<li>
<p>否则，在 <code>tails</code> 数组中，用<strong>二分查找</strong>找到第一个大于或等于 <code>num</code> 的元素，并用 <code>num</code> 替换它。这一步是关键，它并不会增加子序列的长度，但它用一个更小的数更新了某个长度的子序列的最小末尾值，这为后续元素提供了更大的增长潜力。</p>
</li>
</ol>
<p>最终，<code>tails</code> 数组的长度就是最长递增子序列的长度。</p>
<p><strong>JavaScript 代码实现：</strong></p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">/**
 * 进阶解法：贪心 + 二分查找
 * <span class="hljs-doctag">@param</span> {<span class="hljs-type">number[]</span>} nums 整数数组
 * <span class="hljs-doctag">@return</span> {<span class="hljs-type">number</span>} 最长严格递增子序列的长度
 *
 * 时间复杂度: O(n log n)。遍历 nums 为 O(n)，每次遍历中的二分查找为 O(log n)。
 * 空间复杂度: O(n)。tails 数组在最坏情况下可能存储 n 个元素。
 */</span>
<span class="hljs-keyword">const</span> lengthOfLIS = <span class="hljs-keyword">function</span>(<span class="hljs-params">nums</span>) {
    <span class="hljs-comment">// tails 数组。tails[i] 表示长度为 i+1 的所有递增子序列中，结尾最小的那个元素。</span>
    <span class="hljs-comment">// 这个数组是整个算法的核心，它一定是单调递增的。</span>
    <span class="hljs-keyword">const</span> tails = [];

    <span class="hljs-comment">// 遍历输入数组的每一个数字</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">const</span> num <span class="hljs-keyword">of</span> nums) {
        <span class="hljs-comment">// 如果 tails 为空，或者当前数字 num 大于 tails 数组的最后一个元素</span>
        <span class="hljs-comment">// 这意味着 num 可以扩展当前最长的递增子序列，使其变得更长。</span>
        <span class="hljs-keyword">if</span> (tails.<span class="hljs-property">length</span> === <span class="hljs-number">0</span> || num > tails[tails.<span class="hljs-property">length</span> - <span class="hljs-number">1</span>]) {
            <span class="hljs-comment">// 直接将 num 添加到 tails 数组末尾，最长递增子序列的长度+1</span>
            tails.<span class="hljs-title function_">push</span>(num);
        } <span class="hljs-keyword">else</span> {
            <span class="hljs-comment">// 如果 num 不大于 tails 的最后一个元素，我们需要在 tails 中找到一个位置来放置 num。</span>
            <span class="hljs-comment">// 目标：找到 tails 中第一个大于或等于 num 的元素，并用 num 替换它。</span>
            <span class="hljs-comment">// 这样做可以优化未来的可能性：在保持子序列长度不变的情况下，让它的结尾数字变得更小。</span>
            <span class="hljs-comment">// 因为 tails 是有序的，我们可以用二分查找来加速这个过程。</span>
            
            <span class="hljs-keyword">let</span> left = <span class="hljs-number">0</span>;
            <span class="hljs-keyword">let</span> right = tails.<span class="hljs-property">length</span> - <span class="hljs-number">1</span>;
            
            <span class="hljs-comment">// 二分查找的目的是找到插入或替换的位置</span>
            <span class="hljs-keyword">while</span> (left &#x3C; right) {
                <span class="hljs-keyword">const</span> mid = <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">floor</span>((left + right) / <span class="hljs-number">2</span>);
                <span class="hljs-keyword">if</span> (tails[mid] &#x3C; num) {
                    <span class="hljs-comment">// 如果中间值小于 num，说明目标位置在右半部分</span>
                    left = mid + <span class="hljs-number">1</span>;
                } <span class="hljs-keyword">else</span> {
                    <span class="hljs-comment">// 如果中间值大于或等于 num，说明目标位置可能就是 mid，或者在左半部分</span>
                    right = mid;
                }
            }
            
            <span class="hljs-comment">// 循环结束后，left (或 right) 就是第一个大于或等于 num 的元素的位置</span>
            tails[left] = num;
        }
    }

    <span class="hljs-comment">// 遍历结束后，tails 数组的长度就是最长递增子序列的长度。</span>
    <span class="hljs-comment">// 注意：tails 数组本身不一定是一个真实的最长递增子序列，但它的长度是正确的。</span>
    <span class="hljs-comment">// 例如，对于 [10,9,2,5,3,7,101,18]，tails 的变化过程是：</span>
    <span class="hljs-comment">// [10] -> [9] -> [2] -> [2, 5] -> [2, 3] -> [2, 3, 7] -> [2, 3, 7, 101] -> [2, 3, 7, 18]</span>
    <span class="hljs-comment">// 最终 tails 为 [2, 3, 7, 18]，其长度 4 就是最终答案。</span>
    <span class="hljs-keyword">return</span> tails.<span class="hljs-property">length</span>;
};
</code></pre>
<h3 id="解法二动态规划-基础解法"><a href="#解法二动态规划-基础解法">解法二：动态规划 (基础解法)</a></h3>
<p>这是一种更直观的解法，时间复杂度为 O(n2)。</p>
<p>核心思想：</p>
<p>我们定义一个 dp 数组，其中 dp[i] 表示以 nums[i] 这个数结尾的最长递增子序列的长度。</p>
<p>为了计算 <code>dp[i]</code>，我们需要回头看 <code>i</code> 之前的所有元素 <code>nums[j]</code> (其中 <code>j &#x3C; i</code>)。如果 <code>nums[i] > nums[j]</code>，说明 <code>nums[i]</code> 可以接在以 <code>nums[j]</code> 结尾的递增子序列后面，形成一个更长的递增子序列，长度为 <code>dp[j] + 1</code>。我们遍历所有 <code>j &#x3C; i</code> 的情况并取最大值。</p>
<p>最后，整个 <code>dp</code> 数组中的最大值就是最终答案。</p>
<p><strong>JavaScript 代码实现：</strong></p>
<p>JavaScript</p>
<pre><code class="hljs language-js"><span class="hljs-comment">/**
 * 解法二：动态规划
 * <span class="hljs-doctag">@param</span> {<span class="hljs-type">number[]</span>} nums 整数数组
 * <span class="hljs-doctag">@return</span> {<span class="hljs-type">number</span>} 最长严格递增子序列的长度
 *
 * 时间复杂度: O(n^2)，其中 n 是数组 nums 的长度。我们使用了两层嵌套循环。
 * 空间复杂度: O(n)，需要一个大小为 n 的 dp 数组。
 */</span>
<span class="hljs-keyword">const</span> lengthOfLIS_DP = <span class="hljs-keyword">function</span>(<span class="hljs-params">nums</span>) {
    <span class="hljs-comment">// 边界情况：如果数组为空，则最长递增子序列长度为 0</span>
    <span class="hljs-keyword">if</span> (nums.<span class="hljs-property">length</span> === <span class="hljs-number">0</span>) {
        <span class="hljs-keyword">return</span> <span class="hljs-number">0</span>;
    }

    <span class="hljs-comment">// dp 数组定义：dp[i] 表示以 nums[i] 这个数结尾的最长递增子序列的长度。</span>
    <span class="hljs-comment">// 初始化 dp 数组，每个元素至少可以自成一个子序列，所以初始长度都为 1。</span>
    <span class="hljs-keyword">const</span> dp = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Array</span>(nums.<span class="hljs-property">length</span>).<span class="hljs-title function_">fill</span>(<span class="hljs-number">1</span>);

    <span class="hljs-comment">// 从第二个元素开始遍历数组</span>
    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> i = <span class="hljs-number">1</span>; i &#x3C; nums.<span class="hljs-property">length</span>; i++) {
        <span class="hljs-comment">// 对于当前元素 nums[i]，我们需要向前查找所有比它小的元素</span>
        <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> j = <span class="hljs-number">0</span>; j &#x3C; i; j++) {
            <span class="hljs-comment">// 如果找到了一个前面的、比当前元素小的 nums[j]</span>
            <span class="hljs-keyword">if</span> (nums[i] > nums[j]) {
                <span class="hljs-comment">// 说明 nums[i] 可以接在以 nums[j] 结尾的子序列后面，</span>
                <span class="hljs-comment">// 我们需要更新 dp[i] 为 "dp[j] + 1" 和 "当前 dp[i] 值" 中的较大者。</span>
                dp[i] = <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">max</span>(dp[i], dp[j] + <span class="hljs-number">1</span>);
            }
        }
    }

    <span class="hljs-comment">// 遍历完所有元素后，dp 数组中存储了以每个元素结尾的最长子序列长度。</span>
    <span class="hljs-comment">// 我们需要找到这个 dp 数组中的最大值，即为全局的最长递增子序列的长度。</span>
    <span class="hljs-keyword">return</span> <span class="hljs-title class_">Math</span>.<span class="hljs-title function_">max</span>(...dp);
};
</code></pre>
